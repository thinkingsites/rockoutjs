Rockout is a general utility library for Javascript and components for Knockout JS. Most of the Rockout Utils library is intended to be independant of Knockout, for use with any project. The UI library is exclusively for Knockout and dependant on the Utils. It is intended to be friendly with AMD loaders, and NodeJS.
